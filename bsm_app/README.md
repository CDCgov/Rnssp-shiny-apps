# Bayesian Spatio-temporal Modelling Demo App
This R-shiny application allows user to apply to Bayesian Spatio-temporal Models to ESSENCE data through the [epistemic package](https://gitlab.jhuapl.edu/nationalhealth/essence/essence-ai-toolkit/epistemic).  It includes tools to generate basic queries, define custom models, fit models, save models, and visualize model outputs through interactive tables, timeseries plots and maps.

## Documentation

The app's main documentation is rendered from `src/documentation/documentation.md` and opened from the gear menu in the running application.

That page now includes:

- a user guide covering data loading, model fitting, saved artifacts, and visualization tabs
- a technical guide describing the reactive flow, formula construction, feature store, and model-development extension points
